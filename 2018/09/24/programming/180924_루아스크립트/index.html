루아스크립트가 뭐지

2018-09-24 Comments programming

메이플스토리2 소스코드 공개

8월에 메이플스토리2 강화 확률 버그로 인해 공지가 올라왔었다. 그런데 되게 충격적이었던게 어디에서 버그가 있었는지 소스코드 일부를 공개했다! 확률이 민감한 문제라 그런지 정직하게 보이고 싶었던 것 같다!

어쨌든 확률조작이라는 누명은 벗었는데, 코드를 구경하던 중 흥미로운 부분을 발견해따. 확률 계산 부분을 루아스크립트를 이용하는 것이당!

스크립트언어니까 수정이 많은 부분에 적용했구나 하구 유추해봤는데.. 그러고보니 C++과 스크립트언어를 연동하면 어떻게 되는 걸까?? 루아스크립트는 어떤 거지?? 스크립트언어들은 어떻게 실행이 되는거지??.. 모르는게 정말 많다는걸 느낀다ㅠ

C++ 빌드 과정

C++ 에선 어떻게 빌드되는 거지?? 우리는 빌드 과정을

  1. 전처리기(매크로 치환)
  2. 컴파일러(어셈블리어로)
  3. 어셈블러(기계어로)
  4. 링커(라이브러리 연결)

이렇게 배웠다

그러니까 함수는 기계어로 빌드되고, 실행하면 해당 함수로 점프해서 동작할텐데, 스크립트 언어는? 같이 빌드되지 않는다. 스크립트가 수정될 때마다 C++프로젝트를 다시 빌드한다면 의미가 없어지기도 하고..

그러면 스크립트를 호출하는 부분은 어떻게 동작하는 걸까?? 루아 API 에서 스크립트 파일을 읽고, 그걸 적재해서 실행하는 건가??? 먼저 루아스크립트가 무엇인지에 대해 알아봐야 할 것 같다

루아스크립트

인터프리터형 스크립트 언어로 C/C++에 접착시키기 쉽고, 인터프리터 언어기 때문에 재컴파일 없이 바로 설정 변화를 적용할 수 있다. 와우에서 사용하는 거로 유명하다.

루아 API

루아에서는 C++에서 루아 함수를 호출할 수 있도록 API 를 제공한다. 이 API 를 통해서 C++ 에서 루아로 함수 호출, 데이터 전달이 가능하다.

Lua 사이트 에서 C++에서 쓸 수 있는 루아를 다운로드 받을 수 있는데, 소스 형태로 배포된다. 와우! MIT 라이선스이다. 빌드해서 써야한다. 그게 싫다면 여기의 History 에서 버전을 누르면 라이브러리로 다운받을 수도 있다.

다운받거나 빌드한 라이브러리를 C++프로젝트에서 불러온 후에 lua.h 같은 헤더파일을 include 할 수 있다. 여기서 extern “C” 는 루아가 C로 만들어졌기 때문인데, extern “C”에 대한 자세한 의미는 여기에서 알 수 있다. 되게 친절하게 설명해주셨다. 10년 넘게 도움을 주시는 중..
